Reporting a Security Vulnerability

If you discover a security vulnerability in Lovely Beats, please report it responsibly.

- Do not disclose the vulnerability publicly before it has been reviewed.
- Provide a clear description of the issue and, if possible, the steps to reproduce it.
- Include your device model, Android version, and app version.

We will investigate all valid reports and work to resolve confirmed security issues as quickly as possible.

Thank you for helping keep Lovely Beats secure. ❤️
